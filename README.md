# Teaching by Provider Project

Teaching web project containing two app folders:

- `Web_1` - main teaching/member experience
- `Web_2` - MedQuiz static app

## Local Configuration

`Web_1/js/supabase-config.js` is intentionally ignored because it contains local Supabase/API configuration. Copy `Web_1/js/supabase-config.example.js` to `Web_1/js/supabase-config.js` and fill in local values when setting up the project.
