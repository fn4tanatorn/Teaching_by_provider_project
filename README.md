# Web_test_Mini_game

เว็บเกมหมากกระดานแบบ row x column สำหรับให้ผู้เล่นเลือกเส้นทาง เดินได้เฉพาะช่องทางเดิน และต้องตอบโจทย์ประจำช่องให้ถูกก่อนเดินต่อ

## Run

เปิด `index.html` ได้โดยตรง หรือรัน local server:

```bash
python3 -m http.server 5173
```

จากนั้นเปิด `http://localhost:5173`

## Features

- ปรับจำนวน row และ column ได้
- เลือกช่องทางเดินได้
- เพิ่มโจทย์และคำตอบให้แต่ละช่องได้
- ตั้งจุดเริ่มและจุดจบได้
- ผู้เล่นเดินได้เฉพาะช่องที่ติดกัน
- ช่องที่มีโจทย์จะล็อกจนกว่าจะตอบถูก
